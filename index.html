Bootstrap ve ikonları kullanabilmek için öncelikle kurulum yapmalıyız.
Kurulum oldukça basit, bize gereken 3 referans linki var.

  • Bootstrap CSS
  • Bootstrap JS
  • Bootstrap Glyphicon

Öncelikle "https://getbootstrap.com" sitesine girerek "All Releases" kısmından sürümleri görüntüleyiniz. Sıradaki adım için >>

Daha sonra buradan son ya da istediğiniz bir sürümü seçiniz. Karşınıza gelecek sayfada CSS ve JS için gerekli kodlar olacak.

CSS KODU:

CSS için aşağıdaki gerekli kodu "head" bölümüne yazmalısınız.

<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.0.2/dist/css/bootstrap.min.css" rel="stylesheet" integrity="sha384-EVSTQN3/azprG1Anm3QDgpJLIm9Nao0Yz1ztcQTwFspd3yD65VohhpuuCOmLASjC" crossorigin="anonymous">

JS KODU:

JS için aşağıdaki gerekli kodu "body" kısmında sona yazmalısınız.

<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.1.3/dist/js/bootstrap.bundle.min.js" integrity="sha384-ka7Sk0Gln4gmtz2MlQnikT1wXgYsOg+OMhuP+IlRH9sENBO0LRn5q+8nbTov4+1p" crossorigin="anonymous"></script>

GLYPHICON KODU:

İkon için aşağıdaki gerekli kodu "head" kısmına yazmalısınız.

<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.4.0/css/bootstrap.min.css">
